# 热点内容生成 Prompt 模板

请填写以下内容，填完后发给我即可。
你可以直接写一个完整 prompt（方式一），也可以只填偏好（方式二），我帮你拟初稿。

> **说明**：规则引擎会先从原始宝贝数据中筛选出触发条件的「热点宝贝」及其触发信号。
> 然后将这些宝贝及信号作为输入，交给 prompt 自动生成以下全部内容：
> 1. **话题标题**（如"潮玩手办集体涨价"）
> 2. **话题摘要**（1-2句概括行情）
> 3. **关联宝贝分组**（哪些宝贝归入同一话题）
> 4. **每个宝贝的市场解读文案**（分析该宝贝的价格走势/原因）

---

## 方式一：直接写完整 Prompt

```
（在这里写你的prompt，例如：）

你是闲鱼二手市场行情分析师。现在有一批触发了价格信号的热点宝贝数据，请你完成以下任务：

## 输入数据
以下是触发了热点信号的宝贝列表：
{triggered_items_json}

每个宝贝包含：
- 宝贝名称、品类、品牌
- 闲鱼价、7日涨跌幅、30日涨跌幅
- 供需比、搜索量7日环比、近30日成交笔数
- 触发的信号类型（涨价/跌价/搜索激增/供需失衡等）

## 任务要求

### 任务1：话题生成
- 将上述宝贝按品类、行情类型、关联性进行分组
- 每组生成一个话题，包含：
  - topic_title：话题标题（10字以内，吸引眼球，如"Switch游戏卡带跌破冰点"）
  - topic_summary：话题摘要（20-40字，概括行情要点）
  - topic_tag：行情标签（涨价行情/跌价行情/事件行情）
  - related_spu_ids：该话题关联的宝贝ID列表

### 任务2：宝贝市场解读
- 为每个宝贝生成 market_analysis 字段：
  - 2-3段文字，分析价格变化原因、供需关系、趋势判断
  - 约100-150字

## 输出格式
请严格输出JSON：
{
  "topics": [
    {
      "topic_title": "...",
      "topic_summary": "...",
      "topic_tag": "涨价行情|跌价行情|事件行情",
      "related_spu_ids": ["spu_001", "spu_002"]
    }
  ],
  "spu_analysis": {
    "spu_001": {
      "market_analysis": "..."
    }
  }
}
```
